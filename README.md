# :alarm_clock:Turn-Off-The-Pc(ShutPC) # 
## ℹ️Описание ## 
Приложение, таймер, который выключает ПК по определенному времени. Пока из функционала: 
+ Возможность изменять счет времени(По минутам, по секундам. Сделано так, потому что отображается время только в 2 единицах времени).
+ Возможность изменять тему, цвет приложения из предложенных.
+ Возможность сохранять выбранные настройки.
## :exclamation:Проблемы ##
+ Кнопка "Оставновить/Возобновить" работает с задеркой в 0.2 секунды. Сделано это для того, что пользователь случайно не смог несколько раз нажать на кнопку, в ином случае появляется баг, функционал приложения ломается.
